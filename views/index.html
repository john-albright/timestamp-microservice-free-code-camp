Timestamp Microserive

Project Description

This is project 1 of 5 for FreeCamp.org's APIs and Microservices certificate, the 5th of 6 full stack JavaScript certificates. To use this microservice, access the subdirectory /api for the current UTC time and its unix conversion. If the user would like to convert a UTC time or unix time into its counterpart, he or she must access the subdirectory /api/ followed by the subdirectory with the time/date to be converted. When entering a date and/or time, the date string used in the subdirectory must be that which contains no spaces. To read more about date strings, see MDN's documentation on the Date() constructor. As noted there, "Date objects contain a Number that represents milliseconds since 1 January 1970 UTC." Some examples of proper use of formatting are shown below.

Example Usage

Accessing December 25, 2015 00:00 GMT

  • From date: 2015-12-25
  • From unix time: 1451001600000

Accessing other dates and times

  • Current date & time (UTC/GMT)
  • 1970-05-31
  • 1995-12-17T13:24:00*

*This time will be treated as being within the time zone of the user and will be converted to GMT or UTC.
